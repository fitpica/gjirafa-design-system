Getting Started

Installation

Install the Gjirafa Design System in your project. Works with React, Vue, and plain HTML.

NPM
Hide code 
npm install @gjirafa/design-system
Import in your project
Hide code 
/* CSS — import the compiled stylesheet */
@import '@gjirafa/design-system/dist/gjirafa-ds.css';

/* SCSS — import source for access to variables and mixins */
@use '@gjirafa/design-system/src/index' as *;
CDN (Quick Start)
Hide code 
<link rel="stylesheet" href="path/to/gjirafa-ds.min.css" />
Theme Setup
Theming is done via the data-theme attribute. Set it on <html> or any parent element. CODEX is the default.
Hide code 
<!-- CODEX theme (default) -->
<html data-theme="codex">

<!-- VP Player theme -->
<html data-theme="vpplayer">

<!-- Mixed: VP Player for a section -->
<div data-theme="vpplayer">
  <button class="gds-btn gds-btn--primary gds-btn--md">Orange button</button>
</div>
React Example
Hide code 
import '@gjirafa/design-system/dist/gjirafa-ds.css';

function App() {
  return (
    <div data-theme="codex">
      <button className="gds-btn gds-btn--primary gds-btn--md">
        <span className="gds-btn__label">Get started</span>
      </button>
    </div>
  );
}

Guidelines
Size: Always use GDS icon tokens (--gds-icon-sm, --gds-icon-md, --gds-icon-lg). Never hardcode pixel values.
Color: Icons use currentColor by default — they inherit text color from their parent.
Stroke: Keep stroke-width="2" for consistency. Do not use filled variants.
Alignment: Use .gds-btn__icon or .gds-input__icon wrappers for automatic alignment inside components.

Accessibility
  • Focus ring: 0 0 0 2px var(--btn-focus-offset), 0 0 0 4px var(--btn-focus-ring)

  • Lucide icons at 1.5px stroke width, sized per button (12px-20px)

Show code 
// Timer countdown example
const DURATION = 5; // seconds
const CIRCUMFERENCE = 2 * Math.PI * 12.5; // ~78.5

function startToastTimer(toastEl) {
  const circle = toastEl.querySelector('.gds-toast__timer-ring circle');
  const label  = toastEl.querySelector('.gds-toast__timer span');
  let remaining = DURATION;

  circle.style.strokeDasharray = CIRCUMFERENCE;
  circle.style.strokeDashoffset = 0;

  const interval = setInterval(() => {
    remaining--;
    label.textContent = String(remaining).padStart(2, '0');
    const offset = ((DURATION - remaining) / DURATION) * CIRCUMFERENCE;
    circle.style.strokeDashoffset = offset;

    if (remaining <= 0) {
      clearInterval(interval);
      toastEl.classList.add('gds-toast--dismissing');
      setTimeout(() => toastEl.remove(), 250);
    }
  }, 1000);
}
